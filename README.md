# APT 3065 — USIU Campus Navigator

Final year project report for **APT 3065** at United States International University Africa (USIU-Africa).

The project is an Android application that provides campus navigation, a facilities directory with real-time open/closed status, and emergency contact integration for students and visitors.

- **Platform:** Android (Java, XML)
- **Maps:** Google Maps SDK
- **Storage:** SharedPreferences + Gson (local-first, offline)
- **UI:** Material Design 3
