# 3.3 Specific Aims and Objectives

The specific aims and objectives of this project are to:

1. Conduct a literature survey on existing cafeteria and food ordering systems in order to identify their strengths and weaknesses, and to establish the gap that justifies the development of a dedicated mobile ordering system for USIU-Africa.

2. Design and develop a native Android application for students and cafeteria staff, and a web-based admin panel for administrators — together providing real-time order placement and tracking, advance meal scheduling with recurring options, automated wallet deduction via Supabase Edge Functions, an in-app wallet with staff-managed top-up and deduction, role-based access for three user types, and admin-controlled menu management and staff account creation — all underpinned by a secure, cloud-based Firebase architecture.

3. Test and evaluate the prototype cafeteria ordering system in order to assess the extent to which it resolves the queuing, menu visibility, pre-ordering, payment, and staff demand visibility problems identified in Chapter One.
