# 4.5.2 Software Requirements

The following software tools and platforms will be used in the development of the USIU Cafeteria Ordering System:

- **IDE:** Android Studio (latest stable release) — primary development environment for Java + XML Android development
- **Language:** Java (Android application), JavaScript/Node.js (Firebase Cloud Functions)
- **UI Framework:** Material Design 3 components via AndroidX libraries
- **Version Control:** Git & GitHub for source code management
- **Firebase Services:** Firestore (real-time database), Firebase Authentication (user management), Firebase Cloud Messaging (push notifications), Firebase Cloud Functions (scheduled cron jobs)
- **Design Tools:** Figma or draw.io for wireframes and system diagrams
- **Testing:** Android Emulator (functional testing), physical Android device (performance and FCM testing)
- **Project Management:** GitHub Projects or similar for task tracking
