# 4.5.4 Budget Requirements

The budget for the USIU Cafeteria Ordering System is minimal. All primary development tools are free, and the Firebase infrastructure operates within the free tier limits at prototype scale.

- **Firebase Blaze Plan:** Required to deploy Cloud Functions. No fixed monthly cost — charges only apply if usage exceeds the free tier thresholds (125,000 Cloud Function invocations/month free). At prototype scale this cost is zero.
- **Android Studio:** Free (no license cost).
- **GitHub:** Free for public and private repositories at student scale.
- **Design Tools:** Figma free tier sufficient for wireframes and diagrams.
- **Physical Android Device:** Required for FCM and performance testing — an existing personal device is sufficient; no additional purchase needed.

Total estimated cost: **KES 0** for a fully functional prototype.
