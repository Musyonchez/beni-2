# CHAPTER 5: SYSTEM ANALYSIS AND DESIGN — Overview

This chapter presents the system analysis and design of the proposed USIU Cafeteria Ordering System. It outlines the structural and functional models that define how the system operates, interacts with users, and achieves its intended objectives. The chapter introduces a series of diagrams — system architecture, use case, entity-relationship, data flow, flowcharts, activity, sequence, and class diagrams — which collectively illustrate the internal workings of the system. Wireframes are provided to showcase the user interface design across all five main screens. By integrating both conceptual and visual representations, this chapter establishes a clear foundation for the implementation phase, ensuring that the ordering, wallet, and pre-order features are both technically sound and user-friendly.
