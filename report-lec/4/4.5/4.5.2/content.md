# 4.5.2 Software Requirements

The following software tools and platforms will be used:

- **Frontend:** React, HTML5, CSS3, JavaScript
- **Backend:** Node.js with Express framework
- **Database:** PostgreSQL (with PostGIS extension for geospatial data)
- **Version Control:** Git & GitHub for source code management
- **Development Tools:** Visual Studio Code (IDE), Postman (API testing), pgAdmin (database management), Mapbox/Google Maps API (for mapping and route generation)
- **Project Management Tools:** MS Project/Project Plan 365 for task tracking, Canva/Figma for system diagrams and wireframes
