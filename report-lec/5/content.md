# CHAPTER 5: SYSTEM ANALYSIS AND DESIGN — Overview

This chapter presents the system analysis and design of the proposed USIU Campus Navigation System. It outlines the structural and functional models that define how the system operates, interacts with users, and achieves its intended objectives. The chapter introduces various diagrams, including system architecture, use case models, flowcharts, data flow diagrams, class diagrams, activity diagrams, and sequence diagrams, which collectively illustrate the internal workings of the system. Additionally, wireframes are provided to showcase the user interface design and navigation flow, offering a preview of the end-user experience. By integrating both conceptual and visual representations, this chapter establishes a clear foundation for the implementation phase, ensuring that the navigation system is both functional and user-friendly.
