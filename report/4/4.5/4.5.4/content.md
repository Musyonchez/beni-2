# 4.5.4 Budget Requirements

The budget for the USIU Campus Navigation System is minimal since most of the development tools and frameworks are open-source. However, some costs may be incurred for:

- Cloud hosting services (e.g., AWS, Heroku, or Render subscription)
- Domain name registration for a custom web address
- Third-party API services (e.g., Google Maps API or Mapbox, if usage exceeds the free tier)
