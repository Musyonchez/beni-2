# 4.5.3 Server Requirements

For hosting and deployment of the Campus Navigation System:

- **Web Server:** Node.js runtime environment hosted on a cloud platform (e.g., AWS, Heroku, or Render).
- **Database Server:** PostgreSQL with PostGIS, hosted on the same cloud service or as a managed PostgreSQL instance.
- **Scalability Considerations:** The server environment should allow for scaling to accommodate an increasing number of users (students, staff, and visitors) without compromising performance.
