# 7.3 User Acceptance Testing (UAT)

User Acceptance Testing (UAT) was conducted to verify that the USIU Campus Navigator met its core functional objectives and offered a seamless experience across all target user groups, including new students, faculty, staff, and campus visitors. The primary aim was to assess whether the system could support accurate real-time navigation, provide instant facility status updates, maintain local data persistence, and ensure stability during offline usage. Testing involved real participants performing common daily tasks within the application to evaluate responsiveness, geospatial accuracy, and overall usability.

The UAT scenarios were designed around the system's primary operational goals. Students successfully located academic buildings and amenities using the interactive map and search filters, while the "Smart Status" logic provided accurate operating hours with helpful visual cues. Users verified that their "Favorite" locations were retained across sessions, confirming the reliability of the local storage. Performance testing under varying network conditions confirmed that the application remained robust and functional even without active internet connectivity. Overall, the system was validated as intuitive, reliable, and aligned with the navigation needs of the USIU community, with only minor UI refinements noted.
